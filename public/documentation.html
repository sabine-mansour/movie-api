MoviesInfo

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method URL parameters Request body data format Respond body data format
Get a list of ALL movies /movies GET None A JSON object holding data about all movies
Get data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title /movies/[Title] GET /movies/:title None A JSON object holding data about a single movie containing a title, description, genre, director, image and featured.

Example:
{ "title": "Black Panther", "description": "Black Panther is a 2018 American superhero film based on the Marvel Comics character of the same name.", "director": "Ryan Koogler", "genre":"Adventure", "image": "https://encrypted-tbn1.gstatic.com/images?q=tbn:ANd9GcQPpcKQ9eWZGxJe6eXyCW91eayLVm4KqruvJz3GP0F2T2w46yKZ", "featured": true }
Get data about a movie genre by name /movies/genres/[Genre] GET /movies/genres/:genre None A JSON object holding data about the genre
Get data about a movie director by name /movies/directors/[Name] GET /movies/directors/:name None A JSON object holding data about the director
Register new user /users POST A JSON object holding data about the user to create.
Example:
{ “name” : “Sabine Mansour", “Username”: “sabmans”}		 A JSON object holding data about the student that was added.
Example:
{ “name” : “Sabine Mansour", “Username”: “sabmans”}
Update a user information /users/[Username] PUT /users/:username None A text message indicating the info that was updated.
Add a movie to Favourites /users/[Username]/favourites POST /users/:username/favourites A JSON object holding data about the movie to add to favourites.
Example:
{ “title” : “Black Panther"}		 A JSON object holding data about the movie that was added to favourites.
Example:
{ “title” : “Black Panther"}
Remove movie from Favourites /users/[Username]/favourites/[Title] DELETE /users/:username/favourites/:title None A text message indicating whether the movies was successfully removed from favourites.
Unregister an existing user /users/[Username] DELETE /users/:username None A text message indicating whether the user was successfully deleted
